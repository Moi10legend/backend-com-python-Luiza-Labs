SQLAlchemy é a principal biblioteca ORM de integração de SGBDs com Python. Para os bancos não relacionais ou NoSQL utilizaremos o Pymongo.
ORM (Object Relational Mapping) é um modelo utilizado para passar a POO da linguagem de programação para o modelo relacional do banco de dados. As vantagens de se utilizar um ORM são:
- Menos código
- Melhor manutenção
- Utilização de conectores
- Indicados para CRUDs
