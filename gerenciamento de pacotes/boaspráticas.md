O Python possui uma série de convenções e boas práticas codificadas em PEPs (Propostas de melhorias do Python). A mais conhecida é a PEP 8, que cobre o estilo de codificação.

As principais recomendações são:

1) Usar 4 espaços para identação
2) Limitar as linhas de código a 79 caracteres
3) Usar nome de variáveis e funções em snake_case e CamelCase para classes.

Para nos ajudar nisso podemos utilizar ferramentas de checagem de estilo como o flake8. Para instala-lo utiliza-se o código: pip install flake8.
E para executa-lo:
flake8 [nome do arquivo]

Já o Black é uma ferramenta de automatização da formatação do código que segue a filosofia do "Formato único".

Isort é uma ferramenta para classificar importações alfabeticamente e separá-las automaticamente em seções.
