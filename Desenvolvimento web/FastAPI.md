FastAPI é um moderno framework web para construção de APIs com Python, baseado em padrões como OpenAPI (anteriormente conhecido como Swagger) e JSON Schema. Foi criado por Sebastián Ramírez e lançado em Dezembro de 2018, visando proporcionar uma ferramenta que permitia um desenvolvimento rápido e fácil e que ofereça um alto desempenho. Utiliza tipagem estática para validar dados e gerar documentação automática.
Pontos positivos do FastAPI:
•	Alto desempenho: Comparado ao Node.js e superior ao Flask.
•	Geração de documentação automática: Suporta Open API para documentação.
•	Validação de dados e serialização: Utiliza Pydantic para validação de dados e serialização automática.
•	Facilidade de uso
•	Suporte a async/await: Suporte nativo para operações assíncronas, o que é ótimo para lidar com operações de IO.
